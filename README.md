## Problem Statement  
Millions of students and early-career individuals across Bharat lack access to personalized, explainable guidance for:

- Government schemes  
- Skill development programs  
- Career pathways  
- Eligibility requirements  
- Skill gap identification  

Most available solutions are static, text-heavy, and not accessible in regional languages.

---

## Solution Overview  
PathWise AI is a multilingual, voice-first AI platform that:

- Extracts structured user profiles from voice/text input  
- Performs explainable eligibility reasoning  
- Identifies skill gaps  
- Compares career pathways using feasibility scoring  
- Generates personalized step-by-step action roadmaps  

The system is built on a scalable, serverless AWS architecture to ensure accessibility and reliability.

## System Flow  

1. User provides voice or text input  
2. Speech is converted to text (if voice input)  
3. User profile data is extracted  
4. Eligibility reasoning and feasibility scoring are performed  
5. Personalized roadmap is generated  
6. Results are displayed on the frontend
